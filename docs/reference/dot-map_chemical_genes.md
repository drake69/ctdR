# Map chemicals to gene IDs and symbols

Map chemicals to gene IDs and symbols

## Usage

``` r
.map_chemical_genes(ctd_data, chemicals_ids)
```

## Arguments

- ctd_data:

  Filtered CTD interaction data frame.

- chemicals_ids:

  Character vector of chemical IDs.

## Value

A list with `entrez` (named list) and `symbols` (data frame).
